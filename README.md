## Project Overview
A professional, fully interactive sales analytics dashboard built with **Looker Studio** using **Sample Data** – a realistic e-commerce dataset of 69 transactions across 9 countries (July 2021 – March 2022).

Ideal for portfolios, client presentations, or teaching data visualization.

## Dashboard Features
- Monthly revenue & AOV trend with visible Black Friday / Cyber Monday spike  
- World choropleth map showing revenue by delivery country  
- Product performance (pie chart + ranked horizontal bar)  
- Coupon/campaign effectiveness table  
- Interactive filters: Date Range • Delivery Country • Coupon Code • SKU • Buyer  

## Key Business Insights from Sample Data
- 73% of total revenue came from just **October & November 2021** (promo-driven)  
- Top product **P010** = **47%** of all revenue  
- **United States** = **67%** of revenue (0% VAT)  
- “cyber2021” coupon family delivered the highest promotional ROI  
- Average order value jumps to **$33+** in non-promo periods → strong full-price demand
